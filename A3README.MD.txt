# Burger Specials - GUI Layout Assignment A3
This project is a Windows Forms GUI layout for the Farm Burger Specials application.

## Features:
- Users can select between **Prime Beef** and **Veggie** options.
- Selection will display an image.
- A confirmation button allows users to confirm their choice.
- The exit button closes the application.

## GUI Components:
- **Buttons**: Prime Beef, Veggie, Confirm, Exit
- **Picture Box**: Displays burger images
